# GeoMCP Chatbot

En AI-drevet chatbot for analyse av norske geo- og plansaksdata.

## Om prosjektet

Spesialisert KI-assistent som bistår kundebehandlere med å analysere byggesaker, arealbruk, reguleringsplaner og kartdata. Leverer strukturerte, faktabaserte rapporter basert på GeoMCP-data (Model Context Protocol).

## Teknologi

- React 19 + Vite
- Google Gemini AI
- Tailwind CSS
- FontAwesome

## Komme i gang

```bash
npm install
npm run dev
```

Opprett `.env` fil med:
```
VITE_GEMINI_API_KEY=din_api_nøkkel
```
